# 🚀 Financial Intelligence - Quick Start (5 Minutes)

## ⚡ TL;DR - Get Started Now

### 1️⃣ Deploy Database (2 min)

Go to **Supabase → SQL Editor → New Query** and copy-paste:

```sql
-- Add total_amount to orders
alter table if exists public.orders 
add column if not exists total_amount numeric(10, 2) default 0;

-- Item Costs Table
create table if not exists public.item_costs (
  id bigint generated by default as identity primary key,
  menu_item_id bigint not null references public.menu_items(id),
  cost_per_unit numeric(10, 2) not null check (cost_per_unit >= 0),
  notes text,
  created_at timestamptz default now(),
  updated_at timestamptz default now(),
  unique(menu_item_id)
);

-- Financial Expenses Table
create table if not exists public.financial_expenses (
  id bigint generated by default as identity primary key,
  title text not null,
  amount numeric(10, 2) not null check (amount > 0),
  type text not null check (type in ('fixed', 'variable')),
  category text,
  description text,
  expense_date date default CURRENT_DATE,
  created_at timestamptz default now()
);

-- Indexes
create index if not exists idx_item_costs_menu_item_id on public.item_costs(menu_item_id);
create index if not exists idx_financial_expenses_date on public.financial_expenses(expense_date desc);

-- Security
alter table public.item_costs enable row level security;
alter table public.financial_expenses enable row level security;

drop policy if exists "public read item_costs" on public.item_costs;
create policy "public read item_costs" on public.item_costs 
  for select to anon, authenticated using (true);

drop policy if exists "authenticated manage item_costs" on public.item_costs;
create policy "authenticated manage item_costs" on public.item_costs 
  for all to authenticated using (true) with check (true);

drop policy if exists "public read financial_expenses" on public.financial_expenses;
create policy "public read financial_expenses" on public.financial_expenses 
  for select to anon, authenticated using (true);

drop policy if exists "authenticated manage financial_expenses" on public.financial_expenses;
create policy "authenticated manage financial_expenses" on public.financial_expenses 
  for all to authenticated using (true) with check (true);

-- Real-Time
alter publication supabase_realtime add table if not exists public.item_costs;
alter publication supabase_realtime add table if not exists public.financial_expenses;
```

**Click "Run"** ✅ Done!

## 🔧 Troubleshooting

### "Cost Management section not showing"
→ **Clear browser cache** (F12 → Network → Disable Cache, then refresh)

### "Select item dropdown is empty"
→ **Make sure menuItems are loaded** (Check Admin → Menu Tab first)

### "Save Cost button does nothing"
→ **Check browser console (F12)** for errors

### "Profit shows 0"
→ **That's normal!** Revenue/COGS are calculated from completed orders. Place a test order first.
